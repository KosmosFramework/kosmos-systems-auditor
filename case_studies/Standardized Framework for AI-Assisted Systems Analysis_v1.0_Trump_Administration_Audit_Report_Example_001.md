3. **Transferability Failure**: The system was optimized for one person's decision-making style; when that person left, coordination collapsed (Jan 6, transition chaos, institutional memory loss)

5. **Framework Contribution**: The analysis reveals need for **Personalist Substitution Factor** in OCF calculations for systems with DA <1.0 but strong loyalty networks
